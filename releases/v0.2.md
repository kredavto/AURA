# Repository v0.2: Foundation Repository

## Status

Planned

## Goal

Create the full Sprint 0 foundation for AURA so the repository becomes navigable, decision-ready, and usable by Claude Code without private chat context or independent architecture decisions.

## Product Context

AURA is a commercial local LLM assistant for mobile devices. It must work autonomously on user devices, including offline use without internet access, and become a daily assistant and personal knowledge base.

## Planned Scope

- Master Index
- foundational ADR set
- Mobile Client Architecture
- Android Architecture
- iOS Architecture
- Shared Core Architecture
- AI Architecture
- Memory Engine
- STT Engine
- TTS Engine
- AI Model Router
- Security Architecture
- updated README
- updated CHANGELOG
- updated ROADMAP

## Out Of Scope

- mobile implementation
- backend implementation
- AI runtime implementation
- production CI/CD
- security implementation

## Definition Of Done

Repository v0.2 is complete when Claude Code can start implementation planning without making major architecture decisions.

Active implementation should still wait for `Repository v0.3: Value Proposition Validation`, where AURA tests market demand with potential users.

## Known Limitations

This release is documentation-first. It does not produce production application code.
